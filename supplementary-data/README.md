# TreeFlow data
---

Christiaan Swanepoel, Mathieu Fourment, Xiang Ji, Hassan Nasif, Marc A Suchard, Frederick A Matsen IV, Alexei Drummond. ["TreeFlow: probabilistic programming and automatic differentiation for phylogenetics"](https://arxiv.org/abs/2211.05220). arXiv preprint arXiv:2211.05220 (2022).


## Description of the data and file structure

There are two data analyses:
* 980 H3N2 sequences 
* 62 carnivore mitochondrial DNA sequences

Sequence alignments, tree topologies, starting values, TreeFlow model definition files, and BEAST XMLs are in the respective subdirectories in the `supplementary-data.zip` archive.

## Sharing/Access information

The sequence alignment for the H3N2 analysis has been removed because of license conflicts. This can be obtained from the supplementary material of the original publication:
Vaughan TG, Kühnert D, Popinga A, Welch D, Drummond AJ. [Efficient Bayesian inference under the structured coalescent](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4207426/). Bioinformatics. 2014 Aug 15;30(16):2272-9. doi: 10.1093/bioinformatics/btu201


The sequence alignment for the carnivores analysis was taken from the [BEAST examples](https://github.com/beast-dev/beast-mcmc/blob/master/examples/Benchmarks/benchmark2.xml).


## Code/Software

This data was generated by the pipeline at [https://github.com/christiaanjs/treeflow-paper](https://github.com/christiaanjs/treeflow-paper).

### Requirements
* [`treeflow`](https://github.com/christiaanjs/treeflow)
* BEAST 2
* RAxML
* [LSD 0.2](http://www.atgc-montpellier.fr/LSD/)
